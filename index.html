% # bins
% # is a command-line pastebin client
 
% echo "howdy" | bins
https://gist.github.com/4f3994c1cb19f3d0f1fa44c21751a3a5
 
% bins -s pastebin some_file
http://pastebin.com/AU3iKbd1
 
% bins -s pastie -m "Never gonna give you up"
http://pastie.org/10874403
 
% bins -i http://hastebin.com/ajogeruqib
Never gonna let you down
